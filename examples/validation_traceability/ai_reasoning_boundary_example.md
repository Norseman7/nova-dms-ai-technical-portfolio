# AI Reasoning Boundary Example

## Allowed AI Use

```text
Question:
Compare validated event outcomes for events near a reference level during rotational market-state windows.

AI may:
- retrieve structured evidence
- summarize sample size and validity
- explain observed correlations
- list uncertainty and diagnostics
- suggest follow-up analysis questions
```

## Disallowed AI Use

```text
AI must not:
- invent missing event records
- recalculate raw source facts without approved logic
- ignore sample-size warnings
- turn correlations into guaranteed predictions
- store recommendations as Brain Store facts
- bypass human review
```

## Portfolio Signal

AI is framed as a reasoning and explanation layer over validated evidence. It is not the system's unverified truth source.
